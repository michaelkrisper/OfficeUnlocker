# File Unlocker

This is a single-page web application that allows you to remove protection from Microsoft Excel (.xlsx) and Word (.docx) files.

## How to Use

1.  Download the `index.html` file to your computer.
2.  Open the `index.html` file in a modern web browser (like Chrome, Firefox, or Edge).
3.  Click the "Choose File" button to select the Excel or Word file you want to unlock.
4.  Click the "Unlock and Download" button.
5.  The unlocked file will be automatically downloaded to your computer.

## Privacy

All processing is done within your web browser. No files or data are ever uploaded to a server. Your privacy is fully protected.

## Technology

This tool is built with plain HTML, CSS, and JavaScript. It uses the [JSZip](https://stuk.github.io/jszip/) library to handle the file processing.
